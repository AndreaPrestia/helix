# ADR-0003 — Use Conflict Impact Atlas as First Dogfooding Project

- Status: Accepted
- Date: 2026-07-17

## Context

Helix requires validation against a substantial real project with backend,
frontend, simulation, data, user, security, deployment and explainability needs.

## Decision

Conflict Impact Atlas is the first project created and governed through Helix.

## Consequences

- Helix requirements are tested by a real application.
- Project context must be retained under `examples/conflict-impact-atlas`.
- Improvements discovered during implementation feed back into Helix.
